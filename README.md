## Installation (ArchLinux)

```bash
sudo pacman -S cmake opencl-headers ocl-icd
```

* NVIDIA: `opencl-nvidia`
* AMD: `opencl-mesa` (open-source Clover implementation) or `rocm-opencl-runtime` (official AMD ROCm stack)
* Intel: `intel-compute-runtime`
* CPU Only: `pocl` (Portable Computing Language)
